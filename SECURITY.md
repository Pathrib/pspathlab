# Security Policy

## Supported Versions
We support the latest release of EasyTinda POS. Older versions may not receive security updates.

## Data Protection
- All sales data is stored locally on the device.
- Offline transactions are encrypted before syncing online.
- No third-party tracking libraries are included.

## Reporting a Vulnerability
If you discover a security issue, please open a [GitHub Issue](https://github.com/yourrepo/issues) or email us at support@easytinda.com.
We will respond within 48 hours.
